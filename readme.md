# Bedroom Light Strip

Parts:

- Arduino Mega 2560 (5v)
- RGB+W+WW LED Strip
- _Power_
  - 24v 5a power supply
  - 24v to 9v power regulator
  - 5v regulator
  - 3.3v regulator
  - 5x 24v Transistors
    - TODO: Type?
  - 5x 1k ohm resistors
- _Sensors_
  - Microphone amp board (3v)
    - TODO 3v or 5v
  - Photo sensor (3v/5v)
  - [Future] Lux sensor (3v)
- WiFi ESP8266 mini d1 board
- Real time clock DS3231
- _User input_
  - LCD screen (5v)
  - Joystick with button (5v)
  - Switch for main power (24v)
